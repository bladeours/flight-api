## General Info
This REST API is a part of Flight Search Engine project. \
I've created this project to get information about flights, I'm using it in
[flight search engine MVC]() project. It uses MySQL database to get data.

* DepartureFlights
* ArrivalFlights

### Flight

#### Flight by ID

GET: `/flight/{id}` \
Return:
* departure airport
* arrival airport
* departure date
* flight time
* all seats
* free seats
* price
* company

Here is sample output:
  \
  `curl localhost:8082/flights/3`

#### flight by airport codes

GET: `/flight/{departureCode}/{arrivalCode}?=date` \
parameters:
* date - date of flight in format "YYYY-MM-DD"

Return:
* departure airport
* arrival airport
* departure date
* flight time
* all seats
* free seats
* price
* company
